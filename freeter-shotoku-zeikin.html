# お金・税金

フリーターの確定申告・所得税ガイド
バイト掛け持ちの申告方法も徹底解説

1. フリーターにかかる税金の基本

フリーターのアルバイト収入にも、所得税と住民税がかかります。ただし一定の控除があるため、収入が少ない場合は実際には支払いが発生しないこともあります。

税の種類課税される条件税率
所得税年収103万円超（給与所得控除＋基礎控除を超えた部分）5〜45%（累進課税）
住民税年収100万円超（自治体によって異なる）約10%

バイト先が1か所の場合、会社が源泉徴収してくれるので自分で申告しなくていいケースがほとんどです。問題は掛け持ちのとき。

2. 確定申告が必要なケース

✅ 確定申告が必要なフリーターのケース
  • アルバイトを2か所以上掛け持ちしていて、合計年収が103万円超
  • メインの勤務先以外からの収入が年間20万円超
  • 副業・フリーランス収入がある
  • 医療費控除・住宅ローン控除などを受けたい

逆に言えば、1か所のみのバイトで年収103万円以下なら確定申告は不要です。

3. バイト掛け持ちの場合の申告方法

掛け持ちしている場合、各バイト先から「源泉徴収票」をもらい、翌年2〜3月の確定申告期間中に税務署またはe-Taxで申告します。

📋 申告に必要なもの

各勤務先の源泉徴収票 / マイナンバー / 銀行口座情報（還付がある場合）/ 印鑑（窓口の場合）

e-Tax（国税庁の電子申告サービス）を使えばスマホでも申告できます。確定申告書の作成も画面の指示に従うだけなので、難しくはありません。

4. 確定申告をしないとどうなるか

申告が必要なのに怠ると無申告加算税・延滞税が発生します。また、住民税の計算が正しく行われず、後から高額の請求が来ることもあります。

⚠️ 注意

税務署は各バイト先からの支払い調書でデータを把握しています。「バレないだろう」は通用しません。申告漏れは必ず対処しましょう。

5. 正社員になると年末調整に変わる

正社員になると、会社が毎月の給与から自動的に所得税を天引きし、年末に過不足を精算してくれます。これが年末調整です。

✅ 正社員になることで楽になること
  • 確定申告が原則不要（会社が代わりにやってくれる）
  • 払いすぎた税金が年末に戻ってくる
  • 住民税も給与から天引きされるので払い忘れがない
確定申告の手間から解放されたいなら正社員へ
